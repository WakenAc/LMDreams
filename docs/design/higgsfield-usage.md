# Política de uso de Higgsfield

## Princípio

Higgsfield pode ser usado como apoio de design/código durante desenvolvimento, mas **não faz parte do runtime** do website final.

## Regras obrigatórias

- Não incluir scripts, SDKs, embeds, iframes ou dependências runtime de Higgsfield.
- Não carregar assets de produção a partir de URLs Higgsfield.
- Não publicar o website final em infraestrutura Higgsfield.
- Não armazenar credenciais Higgsfield no repositório.
- Não apresentar imagens geradas como projetos reais da LMDreams.

## Estado atual do projeto

- website gerado estaticamente com Astro
- assets servidos localmente a partir do repositório
- deploy via GitHub Pages
- nenhuma dependência runtime de Higgsfield
